# ParcelPal Project

## Project stack

> Vue 3 + Composition API approach

> Vite - Modern build tool

> Element Plus - UI Component Library for vue 3

> Post CSS - CSS Processing Tool

> Pinia - state management

> Mirage.js - used in mock api

## HOW TO RUN

> pnpm install

> pnpm dev

## HOW TO BUILD

> pnpm build
